Abstract




  • Image segmentation is the task of associating pixels in an image with their respective object class labels.
  • In this tutorial, you will learn to:
    • Perform image segmentation using the state-of-art deep learning approaches and customized datasets.
    • Build an end-to-end image segmentation pipeline: dataset customization and transformation, model training, validating, and testing, techniques for postprocessing.
