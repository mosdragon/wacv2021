Home Depot's KDD 2020 Workshop

Computer Vision: Deep Dive into Object Segmentation Approaches


Presentation Date: Wednesday, August 25th, 2020

Workshop Resources

Abstract


Image segmentation is the task of associating pixels in an image with their respective object class labels. Many deep learning-based approaches have been developed for image-level object recognition and pixel-level scene understanding — with the latter requiring a much denser annotation of scenes with a large set of objects. This tutorial provides an end-to-end pipeline for performing image segmentation using the state-of-art deep learning approaches and public datasets.


The ADE20K dataset is a densely annotated dataset spanning a total of 3,169 object classes across 1,072 complex everyday scenes. There are a total of 25k images with an average of 19.5 annotated instances and 10.5 annotated object classes per image. Despite the impressive volume and variety of this dataset, efforts have not been made to realize its full potential. The goal of this tutorial is to provide participants with a strong understanding of how to build image segmentation models while unlocking the full potential of the ADE20K dataset for developing practical applications.


The hands-on session will provide instructions for dataset customization, transformation, and training, validating, and applying segmentation models. In addition to ADE20K, various other open-source image datasets including Microsoft COCO, PASCAL VOC2017, and SUN will be introduced. Performances of various state-of-art segmentation models including U-Net, DeepLab, PSPNet, SegNet, and Mask R-CNN will be compared. Codebase written using Jupyter notebooks with comments and visualizations will be provided and details for prerequisite installations will also be included.
